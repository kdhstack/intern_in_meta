# 장바구니 프로젝트

## 프로젝트 개요

- 프로젝트 목표: 장바구니 기능 추가
- 장바구니에 담긴 상품의 개수를 조정하고, 총액을 표시하는 기능
- 구매 상품의 총액에 따라 배송비 추가 여부 결정
- 참여한 팀원: 김동훈

## 개발 환경

사용한 프레임워크, 라이브러리, 언어 등

- React 18.3.1
- Recoil 0.7.7
- json-server 1.0.0-beta.3

## 설치 및 실행 방법
프로젝트를 로컬에서 실행하기 위한 설치 방법과 명령어

모킹 서비스(json-server 사용, MSW 대용): npm run start:server
개발 서버 구동: npm start

* 두 개의 터미널 창을 열어서 모킹 서비스를 먼저 실행한 상태에서 개발 서버 구동 명령어를 실행해야 한다.

## 컴포넌트 구조

주요 컴포넌트들의 구조와 역할
- CartPage - 장바구니 버튼을 눌렀을 때 나오는 페이지
  - 총 결제액에 따른 배송비 추가 여부 결정
  - 전체 구매금액 표시
  - 결제 버튼
- CartItem - 장바구니 속 담긴 물건의 정보를 담는 요소
  - 장바구니에 담긴 물건의 목록과 개수를 조정하는 기능을 제공
  - 물건의 이름, 가격, 이미지 등 정보
- useCartActions = 장바구니 속 물건의 개수를 조정할 때 쓰이는 함수
- ProductsView - 서버에서 상품 정보를 불러올 쓰이는 함수
  - 로딩, 에러 등 표시
