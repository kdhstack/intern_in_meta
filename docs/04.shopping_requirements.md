# 쇼핑몰 연동 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 
슈킹 온라인 쇼핑몰 페이지

### 사용하게 될 기술: 
React
db.json 

### 문서 작성자: 
김동훈

## 📝 고객 요구사항 정리
- 상품 별 상세 페이지 구현 및 연결
  - 관련 상품도 표시될 수 있도록 구현
- 기존 페이지 간 연결 및 결제 기능 구현
- 일관성 확인
  - ex) 장바구니에서 물건 담을 경우 메인 페이지에서도 담겨있도록 일관된 데이터 표시  

## 📋 기능 목록
- `<button className = 'checkout-button' />` (결제 버튼)
- `<img className = 'itemimg' />` (이미지 클릭 시 상세 페이지로 이동할 수 있도록 변경)

## 📄 컴포넌트
- `<ItemInfoPage />` (상품 상세설명 페이지)
- `<CheckoutDone />` (결제 완료 페이지)
